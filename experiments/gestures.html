Browser Compatibility

This demo uses Pointer Events API that works on modern browsers. If you don't see the gestures working, your browser might not support this API.

Gesture Recognition Not Supported

Your browser may not fully support the gesture recognition features used in this demo. Please try a modern browser like Chrome, Firefox, Safari, or Edge for the best experience.
